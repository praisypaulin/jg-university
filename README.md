# JG University Landing Page

A modern, responsive landing page redesign inspired by jguni.in, built as part of the Reducate.ai Full Stack Development internship assignment.

## Tech Stack
- Next.js
- Tailwind CSS

## Features
- Fully responsive (mobile + desktop)
- Smooth animations
- Hero, About, Programs, Testimonials, CTA, Footer sections

## Live Demo
[View Live](https://jg-university-nine.vercel.app)

## Setup
npm install
npm run dev
